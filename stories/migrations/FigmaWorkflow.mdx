## Workflow

1. Open [VDS Figma file](https://www.figma.com/design/ErBiDvqI7wPQKKXWfhC6yl/VDS---Verkada-Design-System)
2. Navigate to **Collection** frame for the full semantic inventory
3. Rebind component fills/strokes to semantic variables (never raw hex)
4. Verify light and dark columns (dark is labeled "Dark Mode Testing")
5. Use Alert Button and Checkbox as reference patterns before other components
