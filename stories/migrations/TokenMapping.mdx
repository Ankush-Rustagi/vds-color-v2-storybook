## Dialect notes

| Legacy prefix | Origin | Migration approach |
|---------------|--------|-------------------|
| `--color-*` | Verity core | Direct mapping to `--text/*`, `--background/*` |
| `--bg-*` | Early Command | Map to `--background/*` |
| `--vc-*` | Verity charts | Map chart-specific vars to `--text/secondary`, `--chart/*` |
| `--vs-*` | Verity shared components | Map to component semantic groups |
| `--vd-*` | Verity docs/internal | Map to `--link/*`, `--text/*` |
