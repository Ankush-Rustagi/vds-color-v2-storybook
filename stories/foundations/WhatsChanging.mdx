# What's Changing

This page is for **eng leads and PMs** scoping the rollout for stakeholders.

## Scope

- **567 semantic tokens** across **47 component groups** in Figma Collection
- Applies to all Command web surfaces across product lines
- Does **not** change product-line accent colors (cameras = cyan, AC = orange, etc.)
- **Does** change shared component tokens: buttons, checkboxes, text, borders, backgrounds

## Visual deltas teams should communicate

1. **Alert/destructive red** shifts from `#DA5959` to `#CB2939`
2. **Disabled gray** unifies to `#B0B6BE`
3. **Surface backgrounds** use `--background/surface-01` (`#F7F9FB` light)
4. **Same color, different read** (a neutral used as stroke vs fill looks different across modes)

## Risk areas

- **Dark mode values still in testing** (expect refinements after team feedback)
- **Legacy CSS var fragmentation** (teams must map their dialect explicitly; see [Token Mapping](/docs/migrations-color-v2-token-mapping--docs))
- **Accessibility regressions** (contrast is an active constraint, not post-launch cleanup)
