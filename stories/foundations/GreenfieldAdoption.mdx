## Rules

1. **Never hardcode hex** (bind to semantic tokens even if npm package is not shipped yet)
2. **Check both backgrounds** (white canvas and surface-01)
3. **Use Figma Collection as source of truth** until tokens ship in `@verkada/verity` (or equivalent)
4. **Do not create team-specific tokens** (request additions through Design System)
